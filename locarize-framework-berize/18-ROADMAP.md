# Roadmap

## Fase 1 — Fundação (Q3 2026)

**Objetivo:** Estabelecer a base tecnológica do ecossistema.

- [x] Documentação do framework (00 a 20)
- [x] Middleware de indicação (ref proxy + cookie `locarize_ref`)
- [x] Progressive Web App (manifest, service worker, installável)
- [ ] Core Platform (auth, multi-tenancy, permissions, billing)
- [ ] Módulo Commerce (catálogo, pedidos, estoque)
- [ ] Design System e componentes base
- [ ] Infraestrutura multi-tenant
- [ ] Sistema de indicação (Referral Engine)
- [ ] Vínculo de cadastro com código de indicação
