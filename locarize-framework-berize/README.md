# Locarize Framework

Framework modular e desacoplado para construção de produtos SaaS do ecossistema Locarize.

## Filosofia

Construir um ecossistema de produtos onde cada módulo é independente, reutilizável e preparado para escala. Um único Core, múltiplos módulos de negócio, infinitas aplicações verticais.
